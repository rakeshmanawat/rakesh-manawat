Diabetes Prediction Model | Machine Learning

Objective: To predict the likelihood of diabetes based on patient health data.

Key Contributions

  • Cleaned and explored a medical dataset of 768 patient records using Pandas and Seaborn.
  • Analyzed feature correlations (Glucose, BMI, Age, Insulin) impacting diabetes outcomes.
  • Built and evaluated Logistic Regression and SVM models with ~76% accuracy.
  • Visualized model performance using correlation heatmaps and confusion matrices.
  • Provided insights for early detection and preventive healthcare recommendations.
